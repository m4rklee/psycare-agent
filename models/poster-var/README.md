# POSTER-Var Local Assets

This directory is reserved for local POSTER-Var experiment assets.

Expected checkpoint path:

```text
models/poster-var/checkpoints/rafdb_best.pth
```

The checkpoint is intentionally not committed. Download the best model from the upstream POSTER-Var release/source referenced in:

- https://github.com/lg2578/poster-var
- https://pmc.ncbi.nlm.nih.gov/articles/PMC12923884/

The experiment service reports `weights missing` until a compatible checkpoint is placed at the expected path.
