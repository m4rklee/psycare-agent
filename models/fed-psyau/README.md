# FED-PsyAU Local Assets

This directory is reserved for local FED-PsyAU experiment assets.

Expected checkpoint path:

```text
models/fed-psyau/checkpoints/dfme_best.pth
```

The checkpoint is intentionally not committed. Download or train compatible FED-PsyAU weights from the upstream project:

- https://github.com/MELABIPCAS/FED-PsyAU
- https://openaccess.thecvf.com/content/ICCV2025/html/Li_FED-PsyAU_Privacy-Preserving_Micro-Expression_Recognition_via_Psychological_AU_Coordination_and_Dynamic_ICCV_2025_paper.html

The experiment service reports `weights missing` until a compatible checkpoint is placed at the expected path.
